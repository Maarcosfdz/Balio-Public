# Design System Documentation

## Theme Overview

This document outlines the core design principles and token values that define our digital product's aesthetic.

## Color Palette

Our color palette is designed to create a vibrant and engaging user experience, built around a fresh and modern feel.

- **Primary Color**: `#00df96` - A lively green, used for primary calls to action, active states, and brand-critical elements.
- **Secondary Color**: `#468364` - A deep, calming green that complements the primary, used for secondary actions, subtle accents, and supportive UI elements.
- **Tertiary Color**: `#ffd465` - A warm, inviting yellow that adds a touch of brightness, suitable for highlights, badges, or alerts.
- **Neutral Color**: `#707972` - A balanced, earthy grey for backgrounds, text, and non-chromatic UI components, ensuring readability and focus.

Our system operates in a `light` color mode.

## Typography

Our typography system prioritizes clarity and a modern aesthetic, using the `Inter` typeface across the board for consistency and legibility.

- **Headline Font**: `Inter` - Used for all major headings, ensuring strong visual hierarchy.
- **Body Font**: `Inter` - Applied to all main content text, optimized for readability across various screen sizes.
- **Label Font**: `Inter` - Employed for labels, captions, and other smaller text elements, maintaining brand voice even in minute details.

## Shape and Form

Our UI elements feature a `moderate` level of roundedness (value: `2`), providing a friendly yet professional appearance without being overly soft or sharp. This default roundedness offers a contemporary feel.

## Spacing

We adhere to a `normal` spacing standard (value: `2`), which provides a balanced amount of whitespace. This ensures a clean interface, good readability, and distinct separation between elements without feeling either too sparse or too cramped.
